What is Pipes?

Pipes is a fairly new chat application with a blazing-fast backend written purely in C, and a frontend written in C (WebAssembly), JavaScript, HTML, and CSS. Pipes is designed to be a secure & user-friendly chat application that has light rules. Create an account, or sign in today!
